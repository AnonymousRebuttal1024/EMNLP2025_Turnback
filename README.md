# Table 4. Disorder Analysis of LLMs in Route Reversal 

This table illustrates the cognitive disorder in LLMs across all difficulty datasets. The following metrics are reported for each city:

- **Rboustness**: The degree of similarity of routes generated by LLMs to each other. This metric can be a side-effect of whether it has mastered the idea of solving the route reversal problem. - **Misalignment**: Although this metric is widely used for other tasks as well, here it gives an indication of the ability of LLMs to understand the task we are proposing. If LLMs are unable to generate “valid routes” based on the question, we assume that they are unable to understand this route reversal task.
- **Confidence**: In line 244 of the paper, we describe the computation of confidence with prior work. Here, confidence is used to express the probability that LLMs believe that the answer it output is correct. The main significance of this metric is to distinguish whether LLMs have actually entered the geographic space to think.
